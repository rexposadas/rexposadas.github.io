Coding 101 - Bacolod Rotary Club

This is page is dedicated to the Bacolod Rotary club coding class.

Requirements for the class

  1. Laptop - We’ll be coding so you need this.
  2. Pen & Paper - You’ll need this to take notes.

Resources

  1. Visual Studio Code - This is the editor we’ll be using. It’s free and open source. This is OPTIONAL. If you prefer a different editor, you can use that.
  2. Github - This is where we’ll be hosting our code. It’s free. You’ll need to create an account. Once you create an account, send me a message with the URL of your profile.
  3. List of homeworks.

Day 2 ( Dec. 8, 2023)

  1. Review of day 1.
  2. Same homework from day 1. Do these exercises

Day 1 agenda (Nov. 1, 2022)

  1. Introduction to software development.
  2. Setting up your laptop (if you brought one).
  3. Intro to JavaScript

Homework

Do this before our next class.

  1. Create an account here: https://www.freecodecamp.org/ . That’s the website that will have the exercises for the class.
  2. At the bottom of that page, you should see a section that says “JavaScript Algorithms and Data Structures”. Click that section. You should be taken to a list of exercises.
  3. Start from the top and work on the exercises until you get stuck. Then message me in Facebook to let me know which section you were able to do last.

You should be able to do at least 3 of the exercises because of what I covered in the class.

Homework (Extra credit)

Do this only after you are done with the mandatory exercises.

  1. Create an account in https://github.com/
  2. Send me your github name so I can follow you.
